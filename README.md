# AWS EC2 Nginx Web Server

This project demonstrates how to deploy a live web server using **AWS EC2, Linux, SSH, and Nginx**.

The goal of this project was to understand the fundamentals of cloud infrastructure, remote server management, and web server deployment.

---

## Technologies Used

- AWS EC2
- Linux
- Nginx
- SSH
- HTML / CSS
- Git & GitHub

---

## What This Project Demonstrates

- Launching and configuring an EC2 instance
- Connecting to a Linux server using SSH
- Installing and configuring Nginx
- Hosting a static website on a cloud server
- Managing files on a remote machine
- Version control using Git and GitHub
